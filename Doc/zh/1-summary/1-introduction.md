# FastWS-PHP 简介

- FastWS是Fast WebService的缩写. 旨在提供高效稳定的由纯PHP开发的多进程WebService.
- FastWS可以轻松构建在线实时聊天, 即时游戏, 视频流媒体播放, RPC, 以及原本使用HTTP的接口/定时任务的场景中等.
