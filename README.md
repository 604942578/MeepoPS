# FastWS-PHP
FastWS是Fast WebSocke的缩写，志在提供稳定的WebSocket服务。本项目为PHP的语言实现。可以轻松构建在线实时聊天、即时游戏、视频流媒体播放等。

版本V0.0.1 开发中
